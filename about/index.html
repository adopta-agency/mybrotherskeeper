About This Project


This project makes the data associated with President Obama's My Brother's Keeper initiative available in an open and editable format. By using the Adopta.Agency framework - that is, adopting Department of Education datasets - the data that the department has called "statistical indicators" for young men of color will be cleaned up, standardized, expanded upon, and made machine- and human-readable.

This site uses the Adopta.Agency open data blueprint, which is an open Github template designed to be forked and used to improve government and other public datasets.

You can find more information about this project on the Adopta.Agency website.
