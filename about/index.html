About This Project


This is an Adopta.Agency open data blueprint. It is an open Github template desgined to be forked, and used in improviding upon government, and other public data.

Adopta projects look to move forward the open data conversation when it comes to public and private sector data, a conversation that is driven by average citizens.

You can find everything you need to know about this project on the home page, and if you want to know more about Adopta.Agency visit the website, and engage on Github and via Twitter.
