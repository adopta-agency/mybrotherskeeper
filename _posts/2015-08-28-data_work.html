Today I started to download the 130+ datasets that are associated with My Brother's Keeper on the data.gov website.

I'm using Google Docs to upload the files and convert them to CSV files. (The ones I've worked with so far are Excel files.) Then I use Kin Lane's CSV converter tool to turn these into JSON files.

But wow, the data is a mess. Many of the files have weird names so it's not immediately apparent what the data is. They're Excel files, and are created so as to look pretty when printed out, but not to be at all functional when it comes to actually doing anything with the data.

My goal is to work on uploading a dozen or so files a day. Then I'll go back through and clean the data up further.
